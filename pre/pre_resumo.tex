\chaves{Mobile, Web, REST, Restful, Aurelia, JavaScript, NodeJS.}

\begin{resumo}
\textit 
O Orçamento WEB é uma suite multi-plataforma, que integra a plataforma
Web com a mobile através de uma arquitetura
\textit{RestFul}, provendo assim, serviços \textit{rest} na web. Trata-se de um
sistema para criação e acompanhamento de orçamentos e ordens de serviço (O.S)
\textit{on-line} de forma simples, rápida e direta. Para isso, foi criado uma \textit{API}, que disponibiliza os serviços, regras de negócio e acesso aos dados,
desenvolvida utilizando \textit{JavaScript} com NodeJS e banco de dados
PostgreSQL. Um cliente Web, desenvolvido em Html5, Css3 e \textit{JavaScript}
utilizando o \textit{framework} Aurelia em sua versão $1.0.0$. O cliente Web tem
o objetivo, dentre outros, de ser o console administrativo do
sistema, através do uso da \textit{API} criada. Um cliente móvel desenvolvido em Android nativo, que
possibilitará ao cliente da empresa, acompanhar o \textit{status} da Ordem de
Serviço do seu veículo em um centro automotivo. O projeto de software criado
demonstra que é possível criar uma estrutura de integração entre as diferentes
plataformas (Web e Mobile) de maneira homogênea e transparente.
\end{resumo}
